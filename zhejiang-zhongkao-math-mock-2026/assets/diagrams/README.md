# Diagram Types / 图像类型说明

All diagram files follow the naming convention:  
**`{question-number}.{type}.png`**

所有图像文件采用统一命名格式：  
**`{题号}.{类型}.png`**

## Type Suffixes / 类型后缀

| Suffix       | English Meaning        | 中文含义             | Example             |
|--------------|------------------------|----------------------|---------------------|
| `.geo.png`   | Geometry figure        | 几何图形             | `05.geo.png` → Illustration for Question 5 (e.g., triangle, circle) |
| `.func.png`  | Function graph         | 函数图像             | `06.func.png` → Graph for Question 20 (e.g., quadratic function) |
| `.stat.png`  | Statistical chart      | 统计图表             | `20.1.stat.png` → Chart for Question 12 (e.g., bar/line/pie chart) |

> ℹ️ These images are used as visual aids in the main exam document (`zhejiang-zhongkao-math-mock-2026.docx`).  
> ℹ️ 这些图像作为配图，已插入到主试卷文档（`zhejiang-zhongkao-math-mock-2026.docx`）中。
